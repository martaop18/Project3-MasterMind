How To Play?

In <masterMind/> the player must guess a secrete code by clicking the above colored pegs. The goal is to arrange different pegs in the correct order in a row. The resulting pattern must match the hidden code at the top of the board. The player has up to 10 turns to crack the code. It consists of 4 pins, which in turn can be selected from 4 colors for the Beginner level, 5 colors for the Advanced level and six colors for the Expert level. A color can be represented several times.

Clues

The narrow column next to the code preparation area serves as clue row. Clue pegs are either black or white.

A black clue means that there is a correct color in the correct place.

A white clue means that a correct color is in the wrong place.
